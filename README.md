# Scripture Alive

Migrated by Spirit Media Publishing. Built with Astro 5 + Tailwind CSS v4.

## Development

```bash
npm install
npm run dev     # localhost:4329
npm run build   # production build
```

## Pages

- /
- /performances
- /workshops
- /guestpreaching
- /events
- /videos
- /contact

## Stack

- **Astro 5** — pages and static generation
- **Tailwind CSS v4** — styling
- **Cloudflare R2** — media assets
- **Netlify** — hosting

## Deploy

Push to `main` branch → Netlify auto-deploys.
Publish dir: `dist`
Build command: `npm run build`
